* Компонент должен реагировать на клик

Для использования компонента достаточно обернуть им любой узел или текст.

    <A name="zad" onClick={() => {console.log(1)}}>"Link"</A>

Зад

    <A
      name="zad"
      onClick={() => {console.log(1)}}
    >
      "Link"
    </A>
